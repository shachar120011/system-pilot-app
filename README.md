# SystemPilot AI

מערכת AI לניהול אימוץ מערכות, Q&A חכם ומעקב תקלות.

## הרצה מקומית

```bash
npm install
cp .env.example .env.local
# הכנס את VITE_GEMINI_API_KEY ב-.env.local
npm run dev
```

## פריסה ל-Vercel

1. העלה לגיטהאב
2. חבר ב-vercel.com
3. הוסף Environment Variable: `VITE_GEMINI_API_KEY`
4. Deploy
